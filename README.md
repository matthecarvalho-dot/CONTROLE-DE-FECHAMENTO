# Sistema de Controle de Fechamento

Sistema completo para controle de fechamentos de registro de marcas.

## 🚀 Funcionalidades

- 📊 Dashboard comparativo mensal
- 📝 Registro de fechamentos
- 📁 Organização por meses
- 💰 Controle de status (Pago e Assinado / Apenas Assinado)
- 📈 Gráficos de desempenho
- 🔍 Filtros avançados
- 💾 Armazenamento local
- 📥 Exportação de dados

## 🎨 Cores do Sistema

- Azul Escuro: #1a237e
- Amarelo: #ffc107
- Branco: #ffffff

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📦 Instalação

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Comece a usar!

## 🌐 Deploy no GitHub Pages

1. Crie um repositório no GitHub
2. Envie os arquivos para o repositório
3. Ative o GitHub Pages nas configurações
4. Seu sistema estará online!

## 📱 Responsivo

O sistema é totalmente responsivo e funciona em dispositivos móveis.
